# Humanity

Humanity is a game where you survive as an early human between the times of 40,000 BC and 10,000 BC in the Mesolithic.

## Details
This game is made with *Python*, the graphics library being my own raylib bindings (*ultimateraylib*).

## Current features:
* Passive Animals
* Day and Night
